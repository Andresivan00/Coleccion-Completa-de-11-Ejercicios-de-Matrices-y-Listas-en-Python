# Colección Completa de 11 Ejercicios de Matrices y Listas en Python

**El repositorio definitivo** para aprender a trabajar con **matrices (listas de listas)**, **menús interactivos**, **algoritmos clásicos** y **estructura de código limpio** en Python.

### Qué hace este programa
Un **menú principal** con **11 ejercicios reales** de programación, todos 100% funcionales y muy bien estructurados:

| # | Ejercicio | Qué hace |
|---|-----------|----------|
| 1 | Matriz 3×3 con números del 1 al 9 | Rellena secuencialmente y muestra alineada |
| 2 | Matriz 5×n aleatoria | Pide columnas y genera números 0–10 |
| 3 | Suma de dos matrices n×n | Genera A y B → calcula C = A + B |
| 4 | Menú completo 4×4 | Rellenar, sumar fila/columna/diagonales, media (con validaciones) |
| 5 | Matriz 3×3 sin repetidos | Usa `random.shuffle` para números únicos 1–9 |
| 6 | Suma aleatoria de fila o columna | El programa decide al azar qué sumar |
| 7 | Juego del 3 en raya (Tic-Tac-Toe) | Dos jugadores humanos, detección de ganador/empate |
| 8 | Encuesta a 10 personas | Genera datos y calcula porcentajes y sueldos promedio |
| 9 | Matriz 5×5 avanzada | Promedio, mayor, pares, diagonales, ordenamiento con Selection Sort |
|10 | Leer matriz 5×4 desde teclado | Con validación total de entrada |
|11 | Matriz irregular | Cada fila con diferente número de columnas |

### Características destacadas
- Código limpio, comentado y modular (funciones reutilizables)
- Validación robusta de entradas (`pedir_entero`)
- Impresión bonita de matrices alineadas
- Uso correcto de listas por comprensión
- Algoritmos didácticos: Selection Sort, 3 en raya, matrices irregulares
- Todo con **Python puro** – sin librerías externas (solo `random`)
